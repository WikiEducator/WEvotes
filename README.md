# WEvotes

WEvotes is a database-backed tool for wiki-based voting.
These files are part of the
[Widget:Vote](http://wikieducator.org/Widget:Vote)
infrastructure used in scenario planning within
WikiEducator.

## Example / Testing

A playground page is [here](https://wikieducator.org/User:JimTittsler/Voting).
## History

Inspired by the [WEnotes extension](https://github.com/WikiEducator/WEnotes),
Version 1.x of WEvotes stored votes in CouchDB.
Version 2.x stores them in a table in the standard
MediaWiki database.
